Privacy Policy

Last updated: [6 19 2926]

Overview

BonobroGameBot is a Discord party game bot used to run word games, score games, manage lobbies, and send game-related prompts to players.

This Privacy Policy explains what information the bot may collect, how it is used, and how users or server owners can request removal of stored data.


>>Information the Bot May Collect

To run games properly, the bot may temporarily or permanently store:

Discord server IDs
Discord channel IDs
Discord user IDs
Player names or display names
Game lobby information
Current game state
Scores and round results
Player submissions, guesses, clues, votes, and other game-related responses
Custom cards, decks, or settings added by server owners or players

The bot does not intentionally collect sensitive personal information.


How the Information Is Used

The bot uses this information only to provide party game features, including:

Creating and managing games
Tracking players and turn order
Sending private game prompts or cards
Tracking scores
Displaying results and leaderboards
Saving server-specific settings or custom decks
Direct Messages

The bot may send direct messages to users who participate in games. These messages are limited to game-related information, such as secret words, cards, clues, or prompts.

The bot does not use direct messages for advertising or unrelated messages.


Message Content

The bot is designed to work mainly through slash commands, buttons, menus, and forms. It does not need to read unrelated server conversations to function unless a future feature specifically requires it.


Data Sharing

The bot does not sell user data.
The bot does not share user data with advertisers.
Data may be processed by Discord as part of normal Discord bot functionality.


Data Storage

Some game data may be stored while games are active. Some data, such as scores, custom decks, or server settings, may be stored longer if those features are enabled.
Data may be deleted if the bot is removed, reset, updated, or no longer maintained.

Children’s Privacy

This bot is intended for use within Discord servers and follows Discord’s age and account requirements. The bot is not intended to knowingly collect information from children outside of normal Discord bot usage.

Changes to This Policy

This Privacy Policy may be updated from time to time. Changes will be posted on this page with an updated date.
